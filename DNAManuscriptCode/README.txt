Code files used for "Mechanical extrema of DNA and the prevalence of repetitive DNA motifs". 
When running Mainsearch.java include the MATLAB scripts runningforjava.m and runningforjavamolec.m which are MATLAB scripts for JAVA to call to run the Cambridge Model.
One will need to dowload cgDNA+ in order to run this code: https://github.com/rahul2512/cgNA_plus
